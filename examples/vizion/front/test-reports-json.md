## 10. Пузырьковый график: ЖК по объёмам (Bubble Chart)

**Тип графика:** `bubble`

```json
{
  "datasets": [
    {
      "label": "ЖК",
      "data": [
        {"x": 150, "y": 85, "r": 25},
        {"x": 120, "y": 70, "r": 20},
        {"x": 180, "y": 90, "r": 30},
        {"x": 100, "y": 60, "r": 15},
        {"x": 200, "y": 95, "r": 35}
      ],
      "backgroundColor": "rgba(54, 162, 235, 0.5)",
      "borderColor": "rgb(54, 162, 235)"
    }
  ]
}
```

*Где:*
- *X = количество квартир в ЖК*
- *Y = средняя цена (тыс. ₽/м²)*
- *R = размер пузыря = количество продаж*
