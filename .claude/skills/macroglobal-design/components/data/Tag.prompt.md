Status / category pill — PrimeVue Tag severities plus the CRM's deal/pipeline colors.

```jsx
<Tag severity="success" icon="pi-verified">Активен</Tag>
<Tag severity="danger">Просрочено</Tag>
<Tag severity="info">Новый</Tag>
<Tag severity="secondary">VIP</Tag>
```

Severities: `success` `danger` `warning` `info` `primary` `secondary`. Soft tint by default; pass `solid` for filled. Sizes `sm`/`md`/`lg`.
