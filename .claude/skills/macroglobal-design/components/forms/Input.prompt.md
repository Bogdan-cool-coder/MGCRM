Single-line text input matching the CRM's PrimeVue InputText — navy focus ring, optional leading icon.

```jsx
<Input icon="pi-search" placeholder="Поиск по названию" fullWidth />
<Input value="ООО Ромашка" />
<Input invalid placeholder="Обязательное поле" />
```

Props: `icon` (PrimeIcons class), `size` sm/md/lg, `invalid`, `disabled`, `fullWidth`.
