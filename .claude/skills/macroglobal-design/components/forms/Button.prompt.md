Primary action button for the MACRO Global CRM — navy primary, outlined neutral secondary, red danger; pair with PrimeIcons.

```jsx
<Button variant="primary" icon="pi-plus">Создать сделку</Button>
<Button variant="secondary" outlined icon="pi-search">Поиск и фильтр</Button>
<Button variant="danger" text icon="pi-trash">Удалить</Button>
```

Variants: `primary` (brand navy), `secondary`, `danger`. Modifiers: `text`, `outlined`, `loading`, `disabled`, `fullWidth`. Sizes: `sm` / `md` / `lg`. Icons take a PrimeIcons class like `pi-plus`.
