# Раздел «Контакты» — ТЗ для реализации (Claude Code)

Редизайн списочного раздела CRM с переключением **Компании ⇄ Физлица**. Эталон —
`redesign/contacts.html`. Документ описывает разметку, размеры, токены и поведение так,
чтобы фронтендер (или агент `frontend-specialist`) собрал страницу пиксель-в-пиксель.

- **Стек:** Vue 3 + PrimeVue 4 (в проекте), либо React — компоненты ниже даны структурно.
- **Где в коде:** `front/src/pages/ContactsPage/` (`index.vue` + `index.ts`; роут `name: 'Contacts'`
  в `front/src/router/routes/base.ts` лениво грузит `@/pages/ContactsPage`). Состояние колонок/
  фильтров/пагинации — `front/src/pages/ContactsPage/composables/useContactsView.ts` (сохраняется
  per-user в `localStorage`). *(as-built 2026-06-27: исправлен путь — каталога `pages/crm/` нет,
  страница лежит в `pages/ContactsPage/`.)*
- **Шрифт:** `--mg-font-sans` (SF UI Display → web-фолбэк Inter). Иконки — **PrimeIcons 7**.
- **Темы:** светлая и тёмная, переключаются классом `.surface` / `.surface.dark` (см. токены).

## 0. Токены поверхности (light / dark)

Объявляются на корневом `.surface`. Всё остальное ссылается только на эти переменные —
никаких хардкод-цветов в разметке.

| Переменная | Light | Dark |
|---|---|---|
| `--c-page` | `#F1F2F3` | `#272829` |
| `--c-card` | `#FFFFFF` | `#444547` |
| `--c-border` | `#E3E4E6` | `#616263` |
| `--c-border2` (поля/кнопки) | `#D5D6D8` | `#54595E` |
| `--c-text` | `#272829` | `#F9FAFB` |
| `--c-text2` | `#616263` | `#E3E4E6` |
| `--c-muted` | `#7E7F82` | `#9B9C9F` |
| `--c-hover` | `#F9FAFB` | `#3a3b3d` |

Бренд/радиусы — из дизайн-системы: `--mg-primary-900` (акцент, по умолч. `#172747`),
`--mg-primary-950` (navy-полоса), `--mg-primary-800` (hover primary), `--mg-primary-100`
(светлая плашка), `--mg-radius-sm/md/lg`, тени `--mg-shadow-sm/md/lg`.

## 1. Каркас страницы

```
.surface (minHeight:100vh; background: var(--c-page))
├─ navy-полоса (sticky, top:0, z:5)            ← служебная шапка превью
└─ .container (padding:24px 28px; max-width:1340px; margin:0 auto)
   └─ .card (bg:--c-card; border:1px --c-border; radius:--mg-radius-lg;
             shadow:--mg-shadow-md; overflow:hidden)
      ├─ Шапка раздела (§2)
      ├─ KPI-лента (§3)
      ├─ Панель фильтров (§4, раскрывается)
```

## 2. Шапка раздела

Flex-ряд, `align-items:center; gap:12px; padding:16px 20px; border-bottom:1px --c-border;
flex-wrap:wrap`. Слева направо:

1. **Иконка** `pi-building` (компании) / `pi-users` (физлица) — 20px, цвет `--mg-primary-900`.
2. **Заголовок** «Контакты» — `<h1>`, 20px / 600 / `--c-text`.
3. **Переключатель сущности** (segmented), сразу справа от заголовка:
   - контейнер: `inline-flex; gap:2px; background:--mg-gray-100; border-radius:7px;
     padding:3px; **height:38px**`.
   - кнопки «Компания» / «Физлицо»: 13px/600, `padding:7px 16px`, radius 5px. Активная —
     `background:--c-card; color:--mg-primary-900; box-shadow:--mg-shadow-sm`; неактивная —
     прозрачная, `color:--c-muted`.
4. **Поиск** — `position:relative`, иконка `pi-search` (13px, `--c-muted`) абсолютно слева
   (left:11px, центр по вертикали). Инпут: `width:240px; **height:38px**; padding:8px 12px 8px
   32px; border:1px --c-border2; radius:--mg-radius-md; font-size:14px; box-sizing:border-box`.
   Плейсхолдер: компании — «Поиск по названию, БИН…»; физлица — «Поиск по имени, телефону, email…».
5. **Кнопка «Фильтры»** (`Btn`, outlined, icon `pi-filter`) с **бейджем-счётчиком** активных
   фильтров: абсолютный кружок справа-сверху (top:-7; right:-7; min-w:18; h:18; radius:9;
   `background:#E8821E; color:#fff; font-size:10px/700`). По клику — toggle панели §4.
6. **Группа действий**, прижата вправо (`margin-left:auto; inline-flex; gap:8px; align-items:center`):
   - **«⋮» (меню "Ещё")** — icon-only `Btn` outlined, **compact** → **height:31px** (≈18% ниже
     остальных). Меню (§2.1).
   - **«Создать компанию» / «Создать контакт»** — `Btn` primary, icon `pi-plus`, **height:38px**.
   - Группа переносится **целиком** и остаётся справа на узких экранах (не «плывёт»).

### 2.1 Компонент `Btn` (единый для всех кнопок)
- Высота: обычная **38px**, `compact` — **31px**. `box-sizing:border-box`.
- `display:inline-flex; align-items:center; justify-content:center; gap:7px`.
- Паддинг: с текстом `0 14px`, icon-only `0 10px`. 13px/600, radius `--mg-radius-md`.
- Варианты: `primary` (`bg:--mg-primary-900`, hover `--mg-primary-800`, текст #fff),
  `outlined` (прозрачный, border `--c-border2`, текст `--c-text2`, hover `--c-hover`),
  `text` (без border/фона).
- ⚠️ Никаких фиксированных `width` на `Btn` — иначе кнопки с текстом схлопываются.

### 2.2 Меню «⋮» в шапке
Дропдаун (`bg:--c-card; border:1px --c-border; radius:--mg-radius-md; shadow:--mg-shadow-lg;
padding:5px; min-width:200px`), пункты-кнопки 13px/500 с иконкой 15px слева, hover `--c-hover`,
разделители — линия `--c-border`:
`Поиск дубликатов (дедуп)` · `Импорт из файла` · `Экспорт в Excel` · `Настроить колонки` · — · `Массовые действия`.

## 3. KPI-лента (по сущности)

Flex-wrap ряд чипов, `gap:8px; padding:13px 20px; border-bottom:1px --c-border`.
Чип `KpiChip`: pill (`radius:999px; padding:6px 13px`), иконка 12px + «label:» (opacity .8) +
**значение** (700). Цвет по `accent`.

**Компании:** Всего `96` (brand) · Клиенты `58` (success) · Категория L `8` (danger) ·
Категория M `21` (amber) · Категория S `67` (success) · Новые за неделю `5` (teal).

**Физлица:** Всего `248` (brand) · Активные `180` (success) · Без касаний 30+ дн `23` (info) ·
Новые за неделю `12` (teal).

> «Активный клиент» (статус договорных отношений) и «Вовлечённость» (живость контакта по
> `last_activity_at`) — **разные сущности**, не дублируют друг друга. VIP-сущности нет.

## 4. Панель фильтров (раскрывается под шапкой)

`border-bottom:1px --c-border; background:--c-hover; padding:16px 20px`.

- **Строка сегментов:** лейбл «СЕГМЕНТЫ» (11px/700, uppercase, `--c-muted`) + pill-пресеты
  (`radius:999px; padding:6px 13px; 13px/600`; активный залит цветом severity, неактивный —
  border `--c-border`): `Мои` · `Активные`(success) · `С открытыми сделками` · `Без задач`(warning)
  · `Дубликаты`(danger). Справа — `pi-times` (закрыть).
- **Три колонки** (`grid; columns: repeat(3,1fr); gap:18px 24px`), заголовки колонок 11px/700 uppercase:
  - **Кто:** Ответственный · Автор · Вовлечённость.
  - **Что (компании):** Тип компании · Категория (L/M/S1/S2) · Страна · Теги.
    **Что (физлица):** Компания · Должность · Источник · Теги.
  - **Когда:** Создан · Последняя активность.
  - Поле = `FField` (лейбл 12px/500 `--c-text2`) + `FSelect` (рамка `--c-border2`,
    radius `--mg-radius-md`, padding 8px 11px, шеврон `pi-chevron-down` справа).
- **Низ:** справа `Сбросить` (text) + `Применить` (primary, icon `pi-check`).

## 5. Таблица

Контейнер таблицы; `table{ width:100%; border-collapse:collapse }`.

- **`<th>` (`Th`):** padding `10px 14px`; 11px/600; uppercase; letter-spacing .04em;
  `color:--c-muted`; `border-bottom:1px --c-border`; `white-space:nowrap`. Выравнивание:
  `left` по умолчанию, `center` или `right` по флагу.
- **`<td>` (`Cell`):** padding `10px 14px`; 13px; `border-bottom:1px --c-border`; nowrap;
  выравнивание left/center/right; `muted` → `color:--c-muted`.
- Ряд: `background:--c-card`, hover `--c-hover`, `cursor:pointer` (клик по ряду — в карточку).

### 5.1 Колонки — КОМПАНИИ
Порядок строго: **Название · Категория · Страна · Сделки · Посл. контакт · Вовлечённость ·
Ответственный · Теги**. Столбца «⋮» в рядах **нет** (редактирование — переходом в карточку).

| Колонка | Выравн. | Рендер |
|---|---|---|
| Название | left | `Avatar`(32, **square**, инициалы) + ссылка-название (600, `--mg-primary-900`) |
| **Категория** | **center** | `Tag` с буквой; severity: L=danger, M=warning, S1=success, S2=info |
| Страна | left | текст `muted` |
| **Сделки** | **center** | если >0 — кружок-бейдж (min-w:22; h:20; radius:10; `bg:--mg-primary-100`; `color:--mg-primary-900`; 12px/700), иначе «—» |
| Посл. контакт | left | дата/относит. время, цвет по «свежести» (см. 5.3) |
| Вовлечённость | left | точка 8px + текст (Высокая/Средняя/Низкая) |
| Ответственный | left | `Avatar`(22) + имя (12px, `--c-text2`) |
| Теги | left | `Tag`-чипы (wrap), либо «—» |

### 5.2 Колонки — ФИЗЛИЦА
Порядок строго: **ФИО · Компания · Телефон · Посл. контакт · Теги · Автор**.
(Вовлечённость и Источник в таблице физлиц **не показываем**; столбца «⋮» нет.)

| Колонка | Рендер |
|---|---|
| ФИО | `Avatar`(32, круг) + 2 строки: имя-ссылка (600) и должность (11px, `--c-muted`) |
| Компания | ссылка (`--mg-primary-900`) |
| Телефон | текст `muted`, формат `+7 701 234-56-78` |
| Посл. контакт | дата, цвет по свежести (5.3) |
| Теги | `Tag`-чипы / «—» |
| **Автор** | `Avatar`(22) + имя. *Автор = кто привёл/создал контакт* (в компаниях — «Ответственный», кто ведёт). |

### 5.3 Цвет «Посл. контакт» (`TR`)
По давности: `g` свежо → `--mg-green-700` (600); `a` тепло → `--mg-orange-700` (600);
`r` давно → `--mg-red-600` (600); `n` нет касаний → `--c-muted` (400, «—»).

### 5.4 Вовлечённость (`ENG`)
high → «Высокая» (текст `#2b6b38`, точка `#A7EFAA`); mid → «Средняя» (`#9b4029` / `#FFB38A`);
low → «Низкая» (`#7E7F82` / `#D5D6D8`). Точка — кружок 8px.

## 6. Подвал / пагинация

Flex, `justify-content:space-between; align-items:center; padding:12px 20px; border-top:1px --c-border`.

- **Слева (gap:16px):**
  - **Выбор размера страницы** — «Показывать:» (13px, `--c-muted`) + кнопка-дропдаун
    (`height` под кнопки; border `--c-border2`; radius `--mg-radius-md`; 13px/600; шеврон
    `pi-chevron-down`). Меню **раскрывается вверх** (`bottom: calc(100% + 5px)`) с опциями
    **50 / 100 / 200** (выбранная — 700). Состояние per-user.
  - Текст «Показано 1–N из M».
- **Справа:** `pi-angle-double-left` `pi-angle-left` · номера страниц (активная — квадрат
  28×28, radius 5px, `bg:--mg-primary-900`, #fff/600; прочие — `--c-text2`) · `pi-angle-right`
  `pi-angle-double-right`. Иконки 13px, `--c-muted`, padding 4px.

## 7. Tweaks (демо-параметры эталона; в проде — настройки темы)
- **Акцент:** `#172747` (navy) / `#2b2f36` (графит) / `#312e81` (индиго) / `#0f4d3a` (изумруд) —
  меняет `--mg-primary-900/800/100` и плитки.
- **Скругление:** Острые / Скруглённые / Мягкие → набор `--mg-radius-*`.
- **Фон полотна:** Прохладный / Тёплый (`#f4f1ec`) / Брендовый (оттенок акцента) + тёмная тема.
